* 口罩耳朵保護
  由於 2020 肺炎疫情，在台灣防疫中心要求國人要保持適當社交距離；若無法保持則應全程佩帶口罩。但是配帶口罩久了耳朵會不舒服，後來有人推出用 A5 紙折出耳朵保護器可以舒解耳朵壓力。我以為白紙綁在後面有些單調，所以用 CAD 在 A4 尺寸的紙上標示圖案及文字印出來，可以增加一點小趣味。
  [[file:./DWG/口罩耳朵保護.jpg]]
* 圖示說明
  口罩耳朵保護器是用 A5 尺寸折出來的，所以我設計 A4 尺寸圖形用印表機印出後，可以對半做成二個。
